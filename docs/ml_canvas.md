“O problema que estamos resolvendo é a alta taxa de churn em uma operadora de telecomunicações.”

A empresa vem enfrentando uma perda acelerada de clientes, impactando diretamente sua receita e previsibilidade de crescimento. Nesse contexto, surge a necessidade de um sistema baseado em dados que permita antecipar quais clientes têm maior probabilidade de cancelar seus serviços.

Nosso projeto se insere como uma solução de TI orientada a dados, permitindo que áreas como marketing e retenção atuem de forma proativa.

Contexto dos dados

“Para isso, utilizamos um dataset de clientes da operadora.”

Origem dos dados: [dataset público / base interna anonimizada / Kaggle Telco Churn Dataset]
Quantidade de registros: [~7.000 clientes]
Quantidade de features: [~20 variáveis]
Tipo de variáveis:
Demográficas (ex: idade, gênero)
Contratuais (ex: tipo de plano, tempo de contrato)
Uso de serviços (ex: internet, streaming)

A variável alvo é binária:

Churn = 1 → cliente cancelou
Churn = 0 → cliente permaneceu
